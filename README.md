# MCMC_chain_code

This repo is where I keep codes for both running MCMC analyses (specifically for 2pcf multipoles), as well as code to visualize the results with corner plots. 
Much of this code was initially written by either Zack Brown or Ben Levi, and my contributions have been with the goal of increasing efficiency or ease of use. 
The code is a work in progress, with MANY improvements planned.
